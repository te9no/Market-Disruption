### 1. プロジェクトセットアップ ✅

## 技術仕様
- **フレームワーク**: boardgame.io v0.50.2
- **フロントエンド**: React 19.1.1 + TypeScript
- **ビルドツール**: Vite 7.0.6
- **テスト**: Jest + ts-jest
- **多人数対応**: Local multiplayer（最大4人）

## ゲーム特徴
- **二重戦略**: 正規ルート vs 転売ルート
- **リスク・リターン**: 転売は高利益だが威厳ペナルティ
- **市場汚染**: 転売により全体価格低下
- **規制システム**: プレイヤー主導の転売規制
- **オートマ**: AI製造業者と転売業者の参加
