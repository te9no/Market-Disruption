🎮 Webブラウザ版マーケット・ディスラプション

## 開発サーバー起動方法

### サーバー側 (Terminal 1)
cd server
npm run dev

### クライアント側 (Terminal 2)  
cd client
npm run dev

## アクセス方法
- クライアント: http://localhost:5173
- サーバー: http://localhost:3001
